Features & Functionality

Everything currently implemented in the UniLogic toolchain.

Compiler (XPile)

Compilation Targets

  • C99 — native binaries via gcc/clang
  • Python 3 — portable scripts with dataclass types
  • JavaScript (ES6) — browser and Node.js
  • LLVM IR — native binaries via LLVM toolchain

Language Features

  • Types: int (8/16/32/64, signed/unsigned), float, double, string, bool, none, complex, array, list, map
  • User types: structs with fields, single inheritance via inherits
  • Control flow: if/else, while, for each, match/default, escape (break), continue
  • Error handling: Result types (int|error), ? propagation, exit(code), exit codes
  • Casting: cast(x, type)
  • FFI: import "lib" function name(...) with variadic support

Compiler Infrastructure

  • Normalization: 4 levels (0–3) — compound expansion, call flattening, boolean splitting, array expansion. Per-file via @norm N
  • Memory builtins: memtake, memgive, memcopy, memmove, memset, address, deref
  • DR settings: memory, safety, types, int_width, concurrency — per-file, compiler-enforced, cross-file conflict detection
  • Semantic checker: scope chains, forward calls, type inheritance validation, field access resolution

VS Code Extension

Editor Features

  • Syntax highlighting — TextMate grammar for all keywords, types, operators, literals, comments
  • Diagnostics — inline errors from lexer, parser, and semantic checker on every save
  • Hover — keyword glossary (56 entries), function signatures, type definitions, variable types
  • Go-to-definition — jump to function, type, or variable declaration; cross-file workspace search
  • Autocomplete — keywords, functions, types, scope-aware variables, foreign imports
  • Signature help — parameter highlighting on ( and , with nested call support

AI Features

  • Explain Structure — AI-powered walkthrough of file structure with line highlighting
  • Explain Execution — AI-powered execution flow walkthrough

Visualization

  • Dependency graph — D3.js force-directed graph of cross-file function calls with DR conflict highlighting
  • Status bar — rotating tips, visible only when a .ul file is active

Project Manager GUI

  • New project — create project with standard directory layout
  • DR configurator — interactive setting of 5 DR directives
  • Build — compile with target selection (C, Python, JS, LLVM IR)
  • DR conflict checker — validate DR consistency across all project files
  • Project tree — file browser showing functions and types per file
