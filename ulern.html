ULERN

UniLogic Learning Enhanced REPL Nexus — the full-featured terminal REPL

What Is ULERN

ULERN is the desktop version of the UniLogic REPL. It runs in your terminal, connects directly to the compiler and interpreter, and includes features that the browser-based REPL can't offer — persistent history across sessions, full call stack inspection, and direct access to the bytecode toolchain.

The web REPLs are great for trying things out. ULERN is for when you're actually building something.

$ ulern

     ∞  UniLogic
     One Language for Everything
     type 'help' for commands

ul> int x = 42
ul> print x * 2
84

─── Variables ───────────
  int x = 42
─────────────────────

Features

Code Entry & Execution

Type UL code, press enter, it runs immediately. Multi-line input auto-detects unclosed blocks — just keep typing until all blocks are closed.

Variable Pane

After every command, see all defined variables with their types and values. Toggle with pane on/off. Always know what state you're in.

Call Stack Visualizer

See the full call stack on errors or on demand with stack. Shows function names, local variables per frame, and where execution stopped.

Persistent History

Arrow up/down to navigate previous commands. History is saved to disk across sessions — close the terminal, come back, your history is still there.

Function Registry

Define functions interactively and they persist for the session. functions lists all declared functions with their signatures and return types.

Error Diagnostics

Syntax and runtime errors show the call stack so you can trace exactly where things went wrong. No guessing, no silent failures.

Direct Compiler Access

ULERN connects to the same lexer, parser, and AST that the XPile compiler uses. What works in ULERN works in production — no subset, no surprises.

Cross-Platform

Runs on Windows, Linux, and macOS. Uses readline on systems that support it, with a fallback for Windows. Same experience everywhere.

Commands

help          — show available commands
exit / quit   — leave the REPL
functions     — list declared functions with signatures
vars          — show all variables (also shown automatically)
stack         — show the current call stack
history       — print command history
pane on/off   — toggle automatic variable pane
clear         — clear all state and start fresh
Web vs Desktop The web REPLs (Base, Performance, Memory, Execution) run a JavaScript-based interpreter in your browser. ULERN runs the real compiler and interpreter natively on your machine. If you need bytecode compilation, transpilation to C or Python, or access to the full standard library, use ULERN.

Download

ULERN is included in the UniLogic SDK. It requires Python 3.11+ (for the compiler frontend) and runs as a standalone terminal application.
