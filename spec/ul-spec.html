UniLogic Language Specification

Version 1.1 — 2026-03-18

This document is the formal specification of the UniLogic (UL) programming language. It defines the lexical structure, type system, declarations, expressions, statements, runtime directives, annotations, built-in functions, standard library, and formal grammar. A conforming implementation must support all constructs described here.

For the markdown version of this document, see ul-spec.md.

1. Lexical Structure

UL source files are UTF-8 encoded. All keywords, operators, and identifiers use ASCII. String literals may contain any valid UTF-8.

Line comments: // to end of line. No block comments. Whitespace is insignificant except inside strings.

1.1 Keywords (Implemented)

Control: function end returns return if else while do for each in match default escape continue

Declarations: type object inherits new fixed const import from export

I/O: print prompt   Concurrency: parallel spawn wait lock unlock

Hints: nocache yield yields inline pack   Literals: true false empty

Logical: and or not   Memory: address deref left right memmove memcopy memset memtake memgive

Built-ins: size cast absval exit typeof

Reserved (not implemented): iterate, teleport, portal, killswitch, equals, both1, both0, either1, delta, bitflip, negate, change, constant

Type keywords: int integer float double string bool none complex int8 int16 int32 int64 uint8 uint16 uint32 uint64 array list map arena file ok error some

4. Expressions

Arithmetic: + - * / %. Comparison: == != < > <= >=. Logical: and or not. Cast: cast(expr, type). Index: expr[expr]. Field: expr.field. Call: name(args). Propagation: expr?. Post-increment: x++ x--.

9. Standard Library

Function names use module prefix. Method syntax obj.method() rewrites to module_method(obj).

String: str_len, str_upper, str_lower, str_contains, str_starts_with, str_ends_with, str_trim, str_split, str_replace, str_concat, str_substring, str_index_of

Array: array_len, array_sort, array_reverse, array_contains, array_sum, array_min, array_max, array_get, array_set, array_push, array_pop, array_slice

Math: math_sqrt, math_pow, math_abs, math_floor, math_ceil, math_round, math_min, math_max, math_log, math_log2, math_exp, math_pi, math_e

Map: map_keys, map_values, map_has, map_get, map_set, map_delete, map_size

File: file_open, file_read, file_write, file_close, file_exists, file_readlines

JSON: json_parse, json_stringify

10. Formal Grammar (EBNF)

program        = { dr_directive | annotation | declaration } ;
declaration    = function_decl | type_decl | object_decl | foreign_import | const_decl ;
dr_directive   = "@dr" IDENT "=" IDENT ;
annotation     = "@" IDENT [ "(" annotation_args ")" ] ;
const_decl     = "const" type IDENT "=" expr ;
function_decl  = "function" IDENT "(" [ param_list ] ")" [ "returns" return_type ] body "end" "function" ;
return_type    = type | "(" type { "," type } ")" ;
type_decl      = "type" IDENT [ "inherits" IDENT ] { field } "end" "type" ;
object_decl    = "object" IDENT [ "inherits" IDENT ] { field | method_decl } "end" "object" ;
method_decl    = "function" IDENT "(" [ param_list ] ")" [ "returns" return_type ] body "end" "function" ;
foreign_import = "import" STRING "function" IDENT "(" [ param_list ] ")" [ "returns" type ] ;
param_list     = param { "," param } [ "," "..." ] ;
param          = type IDENT [ "=" literal ] ;
type           = [ "<" ] type_name [ ">" ] | "array" type_name ;
body           = { statement } ;
statement      = var_decl | const_decl | assignment | if_stmt | while_stmt | for_stmt
               | match_stmt | return_stmt | print_stmt | spawn_stmt | wait_stmt
               | lock_stmt | asm_block | "escape" | "continue" | expr_stmt ;
var_decl       = [ "fixed" | "const" ] type IDENT [ "=" expr ] ;
if_stmt        = "if" expr body [ "else" body ] "end" "if" ;
while_stmt     = "while" expr body "end" "while" ;
for_stmt       = "for" "each" IDENT "in" expr body "end" "for" ;
match_stmt     = "match" expr { match_case } [ "default" body ] "end" "match" ;
return_stmt    = "return" [ expr | "ok" expr | "error" expr | "(" expr { "," expr } ")" ] ;
print_stmt     = "print" expr ;
spawn_stmt     = "spawn" expr ;
wait_stmt      = "wait" ;
lock_stmt      = "lock" IDENT body "unlock" IDENT ;
asm_block      = "@asm" "(" IDENT ")" { ASM_LINE } "end" "asm" ;
expr           = pipe_expr ;
pipe_expr      = or_expr { "|>" or_expr } ;
or_expr        = and_expr { "or" and_expr } ;
and_expr       = not_expr { "and" not_expr } ;
not_expr       = "not" not_expr | comparison ;
comparison     = addition { ("==" | "!=" | "<" | ">" | "<=" | ">=") addition } ;
addition       = multiplication { ("+" | "-") multiplication } ;
multiplication = unary { ("*" | "/" | "%") unary } ;
unary          = ("-" | "address" | "deref") unary | postfix ;
postfix        = primary { call | index | field_access | method_call | "++" | "--" | "?" } ;
call           = "(" [ arg_list ] ")" ;
index          = "[" expr "]" ;
field_access   = "." IDENT ;
method_call    = "." IDENT "(" [ arg_list ] ")" ;
primary        = INT | HEX | FLOAT | string_literal | "true" | "false" | "empty"
               | IDENT | "cast" "(" expr "," type ")" | "typeof" "(" expr ")"
               | "[" [ array_body ] "]" | "(" expr { "," expr } ")" ;
array_body     = expr { "," expr } | expr "for" IDENT "in" expr ;
string_literal = '"' { char | escape | "{" expr "}" } '"' ;

12. Semantics

Integer division truncates toward zero. and/or short-circuit. Inner scopes shadow outer. Parameters passed by value. ? in non-result functions prints error to stderr and exits with code 1. fixed variables are immutable. empty, 0, 0.0, "", false are falsy.
